# Fonts

Required file:

```text
fonts/materialdesignicons-webfont.ttf
```

Copy your downloaded MDI webfont here:

```bash
cp /root/config/fonts/materialdesignicons-webfont.ttf fonts/materialdesignicons-webfont.ttf
```

The text font is fetched by ESPHome using `gfonts://Elms+Sans`, so you do not need to store Elms Sans locally unless you want a fully offline build.
